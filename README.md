# ⚙️ Installation

```bash
git clone https://github.com/hunt-with-4bh1/webhawk-api.git

cd webhawk-api

pip install -r requirements.txt
```

Install Playwright Browser

```bash
playwright install chromium
```

Run Tool

```bash
python app.py
```

---

# 🔍 Sample Usage
